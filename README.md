# Uni HW

This repository contains university coursework and projects, organized by subject.  
Each folder represents a subject, and each subfolder is a project or assignment.  

---

## Subjects

### Programming Project
- Projects related to C/C++.

### Web Design
- Projects and assignments on Web pages.

### Intro to Cybersecurity
- CTF challenges and solutions completed for the course.  
- CTFs cover multiple **categories** such as crypto, steg, reversing, pwn, etc.

---


## How to Navigate

- Each subject folder contains its own projects/assignments.
- Each project folder should have a `README.md` with details about the project.
- Screenshots, code, and info for each project (if applicable) are inside the respective folder.
