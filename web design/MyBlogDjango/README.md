# Web App Showcase - Django + HTML/CSS/JS

A small web application built with **Django** and **frontend technologies** to showcase multiple web concepts.  
The app is primarily for demonstration and learning purposes rather than production-ready functionality or styling.

---

## Project Overview

The web app includes:

- **Home Page (Blog-style)**:  
  - Posts with text and images  
  - Search bar to filter posts  
  - Review section for user feedback  

- **About Page (CV-style)**:  
  - Displays personal information  

- **User Registration Form**:  
  - Allows new users to register  

- **Gallery Page**:  
  - Image slider with navigation arrows  

- **Chatbot**:  
  - Simple predefined questions and answers  

- **Admin Panel (Django)**:  
  - Manage users, posts, and reviews  

> The project demonstrates a mix of frontend and backend concepts: HTML/CSS/JS for UI, Django for backend and data management.
