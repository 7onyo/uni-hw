# CTF Tools Reference

This file contains commonly used tools for solving CTF challenges.  
It is organized by category.

---

## General Utilities

| Tool       | Purpose |
|------------|---------|
| `strings`  | Extract printable strings from binaries and files |
| `file`     | Identify file types |
| `hexedit`      | Hex dump and hexdump editing |

## Steganography

| Tool        | Purpose |
|-------------|---------|
| `binwalk`   | Extract embedded files and data from images or binaries |

## Reverse Engineering / Binaries

| Tool          | Purpose |
|---------------|---------|
| `Ghidra`      | Disassemble and decompile binaries |
| `IDA`    | Interactive disassembler |

---

## Exploitation / Pwn

| Tool       | Purpose |
|------------|---------|
| `pwntools` | Python library for interacting with binary challenges remotely |
| `gdb`      | Debugging and analyzing binaries |
