# Intro to Cybersecurity

Welcome to my **Introduction to Cybersecurity** repository.
This repo contains all CTF challenges completed as part of the course, along with my solutions, explanations, and any resources provided with the challenges.

## Repository Structure

The repository is organized to keep challenges, solutions, and resources clearly separated and easy to navigate:

```
intro-cybersec-ctfs/
├── ctfs/ # All CTFs organized in separate folders
├── template/ # Template for creating new CTF directory structure
├── resources/ # Optional resources: tools, scripts, references
│ └── tools.md
└── README.md # This file
```

## CTF Folder Structure

Each CTF folder follows this layout to separate the challenge from the solution:

```
├── ctfs/ # All CTFs organized by category
│ ├── category-name/
│ │ ├── ctf-name/
│ │ │ ├── challenge/
│ │ │ │ ├── description.md
│ │ │ │ └── files/ # Original files: images, archives, binaries, etc.
│ │ │ └── solution/
│ │ │   ├── explanation.md
│ │ │   └── files/ # Solution scripts, screenshots, notes, etc.
│ │ └── ...
│ └── ...
└──
```
